Single-cell libraries are often too small to be sequenced on high-end flowcells. Doing this would result in wasteful sequencing depth, i.e. high sequencing saturation.

To overcome this, multiple libraries are loaded on one high-end flowcell, to distribute the number of reads between libraries and achieve desired sequencing depth.

To gain more control over resulting sequencing depth, we can use low-end flowcell run to estimate library quality and composition, and adjust the proportion of each library in the pool.
